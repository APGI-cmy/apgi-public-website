# APW v0.4 Interaction Model

## Status

Drafted for review.

## Principle

The public interaction model is:

Hover -> Click -> Ask -> Train.

## Meaning

Hover gives a short explanation.

Click gives deeper context.

Ask invites the visitor toward Maturion.

Train routes learning interest toward APGI training or contact pathways.

## Gate

Live public Maturion interaction is not part of this Stage 1 artifact. It requires later technical approval.
