# APW TRS v0.4 Trace Matrix

## Status

Drafted for review.

## Purpose

This trace matrix links FRS v0.4 and FRS v0.4A hardening requirements to TRS v0.4 requirement groups, including the v0.4A TRS hardening addendum.

| FRS intent / requirement group | TRS coverage |
|---|---|
| Static public-site boundary and negative functional requirements | APW-TR-V04-001 to APW-TR-V04-005; negative technical requirements; APW-TR-V04-079 to APW-TR-V04-083; additional negative technical requirements |
| Route and page completeness | APW-TR-V04-006 to APW-TR-V04-010; APW-TR-V04-053 to APW-TR-V04-057 |
| Required content models and card families | APW-TR-V04-011 to APW-TR-V04-017; APW-TR-V04-058 to APW-TR-V04-065 |
| Required page sections and offering composition | APW-TR-V04-018 to APW-TR-V04-025 |
| Hover -> Click -> Ask -> Train behavior | APW-TR-V04-026 to APW-TR-V04-032; APW-TR-V04-066 to APW-TR-V04-071 |
| Accessibility and mobile behavior | APW-TR-V04-033 to APW-TR-V04-039; APW-TR-V04-072 to APW-TR-V04-078 |
| Public-safe knowledge and case-study boundary | APW-TR-V04-040 to APW-TR-V04-044; APW-TR-V04-091 to APW-TR-V04-092; additional negative technical requirements |
| Static performance, metadata, and heading behavior | APW-TR-V04-045 to APW-TR-V04-048; APW-TR-V04-079 to APW-TR-V04-088 |
| Validation expectations and downstream QA | APW-TR-V04-049 to APW-TR-V04-052; APW-TR-V04-089 to APW-TR-V04-093 |

## Gate

This trace matrix does not authorize implementation.
