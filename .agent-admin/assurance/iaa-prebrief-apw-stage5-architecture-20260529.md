## 3. Required IAA Review Lens

The IAA must evaluate:

1. Whether Stages 1-4 support moving into Architecture.
2. Whether the Architecture preserves fully functional delivery controls.
3. Whether route, content, hub, accessibility, responsive, deployment, and evidence architecture are sufficiently defined.
4. Whether Stage 6 QA-to-Red can derive concrete checks from this Architecture.
5. Whether the Architecture avoids runtime implementation and unapproved scope expansion.
6. Whether builder correction is required before CS2 proxy disposition.
