## 3. Required IAA Review Lens

The IAA must evaluate:

1. Whether Stage 8 derives from the approved Stage 3 through Stage 7 package.
2. Whether the plan is specific enough for Stage 9 Builder Checklist.
3. Whether Stage 6 RED checks map to implementation work packages and evidence.
4. Whether Stage 7 PBFAG conditions are carried forward.
5. Whether the public-only boundary is preserved.
6. Whether implementation remains gated.
7. Whether governance role separation is preserved.
